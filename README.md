# GitHub Pages Portfolio

## How to use

1. Create a repository named `YOUR_GITHUB_USERNAME.github.io`.
2. Upload all files in this folder to that repository.
3. Replace `YOUR_GITHUB_USERNAME` in `index.html` with your real GitHub username.
4. Replace `Your Name` with your name.
5. Replace `assets/profile.svg` with your real image, or add your image to `assets/` and update the image path in `index.html`.
6. Open `https://YOUR_GITHUB_USERNAME.github.io` after GitHub Pages finishes deploying.
